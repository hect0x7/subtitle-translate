# YouTube 双语字幕 | Bilingual Subtitles

🎬 一个 Chrome 扩展，为 YouTube 视频添加双语字幕（英文在上，中文在下），类似沉浸式翻译的效果。

## ✨ 功能特点

- 🌐 **双语显示** — 英文原文 + 中文翻译同时显示
- 🎨 **沉浸式风格** — 仿沉浸式翻译，金色中文字幕
- 🔧 **可自定义** — 字体大小、显示风格、语言方向
- 🆓 **免费翻译** — 使用 Google Translate 免费接口，无需 API Key
- 📺 **全屏支持** — 全屏模式下字幕自动适配放大
- ⚡ **智能缓存** — 翻译结果缓存，避免重复请求

## 📦 安装步骤

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 打开右上角 **"开发者模式"**
3. 点击 **"加载已解压的扩展程序"**
4. 选择本项目文件夹

## 🚀 使用方法

1. 打开任意 YouTube 视频
2. 点击视频播放器中的 **CC 按钮** 开启英文字幕
3. 扩展会自动将字幕翻译为中文并双语显示
4. 点击浏览器工具栏的扩展图标可以调整设置

## ⚙️ 设置选项

| 设置项目 | 功能描述 | 备注 |
|------|------|------|
| 字幕大小 | 提供 小 / 中 / 大 三档预设 | 快速切换全局字号 |
| 显示风格 | 沉浸式 (金色) / 经典 (蓝色) | 调整整体视觉方案 |
| 背景不透明度 | 0% - 100% 滑块调节 | 修改字幕半透明黑框亮度 |
| 译文字体 | 支持微软雅黑、思源、苹方等 | 针对翻译文本独立设置字体 |
| 译文字号 | 0 - 40px 精确调节 | 0 表示跟随全局预设档位 |
| 译文字重 | 细 / 常规 / 中 / 粗 | 调节翻译文本的粗细程度 |
| 语言方向 | 源语言 → 目标语言 | 支持多语种互译 |

## 🔧 技术实现

- **字幕获取**: 通过 MutationObserver 监听 YouTube 原生字幕 DOM 变化
- **翻译引擎**: Google Translate 免费 API (`translate.googleapis.com`)
- **字幕渲染**: 在 `#movie_player` 上叠加自定义双语字幕层
- **数据缓存**: 内存缓存翻译结果，最大 2000 条
